💰 Personal Finance Tracker

A modern and user-friendly Personal Finance Tracker that helps users manage their expenses, monitor budgets, and visualize spending habits through interactive charts and summaries.

🚀 Features
Add new expenses with amount and category
Delete expenses instantly
Categorize expenses (Food, Travel, Shopping, Bills, Entertainment, etc.)
Set a monthly budget
View total expenses and remaining balance
Interactive expense visualization using pie charts
Data persistence using Local Storage
Responsive and clean user interface
Real-time expense tracking and updates
📸 Project Overview

Managing personal finances can be challenging without proper tracking. This application provides an easy way to record expenses, organize them into categories, and gain insights into spending patterns.

Users can:

Track daily expenses
Monitor budget usage
Analyze category-wise spending
Maintain expense history even after page refresh
