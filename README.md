# 💰 Personal Finance Tracker

A simple and interactive Personal Finance Tracker built using HTML, CSS, and JavaScript. This application helps users manage their expenses, track spending habits, set budget limits, and visualize financial data through charts.

---

## 📖 Overview

Managing personal finances is essential for maintaining financial stability. This application provides an easy way to record expenses, categorize transactions, monitor monthly spending, and compare expenses against a predefined budget.

The tracker stores data locally in the browser, ensuring that expense records remain available even after refreshing or reopening the application.

---

## ✨ Features

- Add new expenses
- Delete existing expenses
- Categorize expenses
- Set a monthly budget
- View total expenses
- Track remaining budget
- Visualize spending using pie charts
- Real-time updates
- Local Storage support for persistent data

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/personal-finance-tracker.git
```

### Open the Project

```bash
cd personal-finance-tracker
```

### Run the Application

Open `index.html` in your browser

OR

Use VS Code Live Server:

1. Install the Live Server extension.
2. Right-click `index.html`.
3. Click **Open with Live Server**.

---

## 🎯 How to Use

### Step 1: Set Monthly Budget

Enter a budget amount and click **Set Budget**.

Example:

```
50000
```

### Step 2: Add Expenses

Example entries:

| Expense | Amount | Category |
|----------|----------|----------|
| Pizza | 500 | Food |
| Movie | 300 | Entertainment |
| Uber | 250 | Travel |
| Shopping | 2000 | Shopping |
| Electricity Bill | 1500 | Bills |

### Step 3: Monitor Spending

The application automatically updates:

- Total Expenses
- Remaining Budget
- Expense Distribution Chart
